Dependency grammars

  • Theory of syntactic structure
  • Structure is modelled on a directed graph (or tree)
    • Each word in a sentence is a node
    • Dependencies: labelled links between nodes

Universal Dependencies (UD)

  • a project that develops cross-linguistically consistent treebank annotation for many languages
  • annotation scheme is based on an evolution of
    • Stanford dependencies
    • Google universal part-of-speech tags
    • Interset interlingua for morphosyntactic features
  • The general philosophy:
    • provide universal inventory of categories and guidelines to facilitate consistent annotation of similar constructions across languages
    • allow language-specific extensions when necessary
  • http://universaldependencies.github.io/docs/

Treebanks

  • Collection of parsed sentences
  • Consistently adhere to a particular syntactic theory
  • Used for:
    • training statistical parser models
    • corpus linguistics, e.g.:
      • extract a valency lexicon for verbs
      • study the frequency of various syntactic phenomena, e.g.:
        • word order
        • nominal case usage and syntactic function

Objectives

  • Manually annotate 300 sentences
  • Establish annotation guidelines
    • Find out which things need special treatment in Kazakh
  • Train a model and test it to see how well we are able to parse

Experiment

  • Three objectives:
    • Test the treebank in a "real world" scenario
    • Try out different feature combinations
    • Estimate performance on new sentences / unseen data
  • Experimental setup
    • MaltParser: transition based dependency parser
